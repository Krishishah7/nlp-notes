# Sentiment Analysis

Sentiment analysis determines the emotional tone of text such as Positive, Negative, or Neutral.

## VADER Sentiment Analysis
VADER is a rule-based sentiment analysis tool available in NLTK.

It works well for short texts such as reviews, feedback, and social media posts.

## Sentiment Scores
- Positive
- Negative
- Neutral
- Compound

## Compound Score
The compound score is a normalized value between -1 and +1 that represents the overall sentiment of a sentence.

### Interpretation
- ≥ 0.05 → Positive
- ≤ -0.05 → Negative
- Otherwise → Neutral
